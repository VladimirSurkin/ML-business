HTTP Method URI Action
GET /predict Предсказать вероятность по известным данным

Следующие GET параметры обязательны при запросе:

{% for param in params %} {% endfor %}
Название Тип Пример значения Описание

Примеры ответа

Ответ возвращается в JSON формате.

После запроса на предсказание отмены возвращаются поля:

statusintСтатус работы скрипта (0 = что-то сломалось: 1 = все хорошо).
predintПредсказание по оттоку клиента (1 = он уйдёт, 0 = останется).
pred_probafloatПредсказание по оттоку клиента в долях (значение между 0 и 1, 1 = он уйдёт, 0 = останется).
